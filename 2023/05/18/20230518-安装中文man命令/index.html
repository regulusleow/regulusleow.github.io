macOS 安装 manpages-zh

发表于 2023-05-18 分类于 开发工具 Disqus：

在 macOS 上，man 命令用于查看系统中安装的程序的帮助文档。然而，默认情况下，这些帮助文档都是英文的，对于不熟悉英文的用户来说可能会有些困难。幸运的是，我们可以安装 manpages-zh 来获取中文帮助文档。

安装依赖

在安装之前，我们需要先安装一些依赖项，依赖项有三个：cmake，python3，opencc，在 macOS 上我们使用 brew 安装这些依赖。

  2. 查看最新的 tag

  3. 切换到最新的 tag

  4. 编译安装

  5. 添加 zh_CN 到配置文件，本人环境为 zsh，我是将环境添加到 .zshrc 中。

  6. 重启终端尝试 man 命令

结论

安装 manpages-zh 后，我们可以方便地查看中文帮助文档，这对于不熟悉英文的用户来说是非常有用的。
